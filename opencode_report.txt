Schema decisions recorded
-------------------------
Created docs/real_lerobot_schema_decision.md with these decisions:

1. First real writer target is forcevla_13d.
2. forcevla_13d mapping:
   observation.image = external RGB camera.
   observation.wrist_image = TCP/wrist RGB camera.
   observation.state = 13D: ee_pos(3) + ee_axis_angle(3) + gripper_pos(1) + wrench(6).
   action = 7D measured TCP delta.
   task = task_instruction.
   prompt = task_instruction.
3. prompt is added for ForceVLA/OpenPI compatibility.
4. task is kept for readability and LeRobot-style compatibility.
5. Secondary writer target remains doosan_full_25d.
6. doosan_full_25d mapping:
   observation.image = external RGB camera.
   observation.wrist_image = TCP/wrist RGB camera.
   observation.state = full 25D model_state.
   action = measured_action 7D.
   task = task_instruction.
   prompt = task_instruction.
7. Final intended v2.1-style layout uses parquet under data/chunk-000/ and MP4 videos under videos/.
8. First real forcevla_13d parquet columns should be observation.state, action, timestamp, frame_index, episode_index, task_index, index, task, and prompt.
9. Images/videos should not be embedded directly into parquet in the first writer.
10. First video writer should encode MP4 from the controlled image_staging tree.
11. Terminal-padded final frames remain excluded by default.
12. The dataset writer should not generate a fake third camera.
13. Laptop/OpenCode writes code; lab workstation validates ForceVLA loader, normalization statistics, and training compatibility.

Skeleton overwrite behavior
---------------------------
Updated write_lerobot_skeleton behavior:

1. If output directory exists and --overwrite is not set, the writer fails clearly before writing.
2. If output path exists and is not a directory, the writer fails clearly.
3. If output directory exists and --overwrite is set, the writer removes only that skeleton output directory with shutil.rmtree(output_root), then writes a clean skeleton.
4. Tests cover both no-overwrite failure and overwrite success.
